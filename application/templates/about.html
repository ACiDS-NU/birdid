Plover

A simple bird identification web app for
new and intermediate birders.

A large flock of Sandhill Cranes landing at Jasper-Pulaski FWA, Indiana. © D. Hsu

How it started

I always wanted to shoot wildlife photos. However, it wasn't until I bought my Sigma 150-600mm tele lens for a trip to Yellowstone NP that I started seeing all the birds. It is amazing how many bird species are around us, yet for a new birder I couldn't recognize them. Although I was using Merlin (an incredibly helpful app) when in doubt, I decided to make an app just to learn more about these beautiful birds.

How it works

The underlying model of this app is a convolutional neural network. It is particularly suitable for analyzing images due to its way of perceiving data. I took a pre-trained model called MobileNetV2, which is capable of identifying 1000 categories of items, as the feature extractor, and trained it on the NAbirds Dataset (36k train images of 404 species) for 10 epochs. Then I fine-tuned it for another 20 epochs. All trainings were done using Tensorflow 2.0 on Google Colab with GPU backend. For the code, see the GitHub page for this project.

When you enter both the date and the location of the observation for your photo, you create a geo-spatial filter for the results. It is done in real-time by querying the Global Biodiversity Information Facility - eBird Observation Dataset (GBIF - EOD). If the candidate bird was (almost) never reported, it is removed from the list. This feature helps distinguish similar species that occur differently in time or range. Geospatial occurance provides two values: (1) Within the 14 days of your observation date, how many times this bird has been recorded in this area, and (2) within the same period and in the same area, how many records are there. This provides a rough estimation of how frequent the bird is logged.

The NAbirds Dataset

This dataset is collected and published by the Cornell Lab of Ornithology, containing ~48k images of 404 species in 555 visual categories. I split them to 36k/6k/6k train/validation/test images randomly, after grouping the visual categories into species. Using the bounding box information provided, I cropped and padded black sides to the images.

This website

This website uses Flask and is hosted on Amazon Elastic Beanstalk. The template is based on Spatial. The background photo you saw when you entered this site is an endangered Piping Plover. That's why this site is called Plover. The bird description is scraped from Wikipedia with BeautifulSoup and with some manual effort. The idea to create this website is inspired by Lazybirder. The geocoding API is powered by Nominatim.
